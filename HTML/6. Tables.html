Chapter 6: Tables

In HTML, a table is written out row by row.

Basic Table Structure

table - Table tag, tr - table row, td - table data

15 15 30
45 60 45
60 90 90

Table Headings

Table Headings = Row/Column titles. Element/Tag = th. Use it like you would use table data. So top left is left empty. Better to use th for screening and appearances.

Saturday Sunday
Tickets Sold: 120 135
Total Sales: $600 $675

Spanning Columns

To span columns, use the colspan attribute on either th or td element. Like so:

9am 10am 11am 12am
Monday Geography Math Computer Science
Tuesday Gym Home Ec

Spanning Rows

To span rows, use the rowspan attribute on either th or td element. Like so:

ABC BBC CNN
6pm - 7pm Movie Comedy News
7pm -8pm Sport Current Affairs
8pm - 9pm Sherlock Anime
