Strong example

used for strong words like CAUTION or ALERT stuff like not suitable for children under 5 years old. By default browsers will show the contents of strong as bolded.

Emphasis example

indicates emphasis that subtly changes the meaning of the sentence, like I think this is a rock, this sentence hints sarcasm. By defalt browsers will show the contents of emphasis as italicized.

Blockquote and q example

Blockquote is used for longer quotes that take up an entire paragraph. Note how the p elemt is still used inside the blockquote element.

q is used for shorter quotes that sit within a paragraph, adds quotes usually depending on the browser.

Did you ever stop to think, and forget to start again?

As A.A. Milne said, Some people talk to animals. Not many listen though. That's the problem.

Abbreviation and Acronym example

You can define an abbreviation for a specific word in the beginning of the paragraph and then when a user hovers over it they can view the full word. Allowing you to use the abbreviation thereafter.

Prof Stephen Hawking is a theoretical physicist and cosmologist.

You can define an acronym similarly:

NASA do some crazy space stuff

Cite example

This is used to cite something you are referencing from a book, film, research paper etc.

A Brief History of Time by Stephen Hawking has sold over ten million copies worldwide

Definition example

A black hole is a region of space from which nothing, not even light, can escape.

Address example

The address element has quite a specific use: to contain contact details for the author of the page. It can contain a physical address, but it doesn't have to. For example it may also contain phone number or email.

Homer Simpson Address:

homer@example.org

742 Evergreen Terrace, Springfield

Insert, Strike, and Delete Example

The insert element can be used to show the content that has been inserted into the document, while the del element can show the text that has been deleted from it.

It was the worst best idea she had ever had.

The s element indicates something that is no longer accurate/relevant but is still important information.

Laptop Computer:

Was $995

Now only $450
