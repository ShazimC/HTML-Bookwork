Bullet Point Styles

  • list-style-type: allows you to control the shape or style of a bullet point (also known as a marker). It can be used on rules that apply to the ol, ul, and li elements.
    • For an unordered list you can use the following values:
      • none
      • disc
      • circle
      • square
    • For an ordered list(numbered) you can use the following values:
      1. decimal
      2. decimal-leading-zero
      3. lower-alpha
      4. upper-alpha
      5. lower-roman
      6. upper-roman

Images for Bullets

  • list-style-image: url("images/star.png"), can be used on li and ul only. You should use margin to increase the vertical distance between each item if the list image is too big.
      • Example:
    • YOSHA
    • OMAE
    • WA
    • MOU
    • SHINDEIRU

    Positioning the Marker

    list-style-position: lists are indented into the page by default and the list-style-position property indicates whether the marker should appear on the inside or outisde of the box containing the main points.
      This property can take on two values:
    • outside: the bullet sits to the left of the block of text/list.
    • inside: the bullet sits inside the box to the left of the text.
    • NOTE: list shorthand: list-style: style, image, and position in any order.

    Table Properties

    Properties Definitions
    width: to set the width of the table
    padding: to set the space between the border of each table cell and its content.
    text-transform: to convert the content of the table headers to uppercase.
    letter-spacing to set the spacing between the letters.
    font-size: to set the size of the font.
    border-top/border-bottom to set borders above and below the table headers.
    text-align: to align the writing to the left of some table cells and to the right of the others.
    background-color: to set the background color of the table
    :hover to highlight a table row when a user's mouse goes over it.
    empty-cells: to specify whether or not their cells that are empty can should show, hide, or inherit from the element they are in.
    border-spacing: allows you to control the distance between adjacent cells. By default there is a gap. It can have horizontal and vertical gap in pixels
    border-collapse: collapse/seperate

    Styling Submit Buttons

      CSS rules:
    • color: used to change the color of the text on the button.
    • text-shadow: can give a 3D look to the text in browsers that support this property.
    • border-bottom: bottom border of the button.
    • background-color: color of the button. Gradient has been used with background-color here which is covered on page 419.
    • :hover psuedo-class has been used to change the appearnce of the button when the user hovers over it.

    Styling FieldSets & Legends

    • width: used to control the width of the fieldset.
    • color: used to control the color of the text.
    • background-color: color behind the items.
    • border: used to control te appearance of the border around the fieldset and/or legend.
    • border-radius: radius of the corners
