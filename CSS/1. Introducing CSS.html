This is Introduction to CSS

This example uses two documents: the HTML file and a seperate CSS file.

New terms:

Link element: Link tag is used to link to an external document (usually a stylesheet)
Rel attribute: stands for relationship and it specifies the relationship between the quoted document and the current document.
Href attribute: specifies where the CSS file is located.

CSS selectors

  • CSS selectors: allow you to target rules to specific elements in an HTML document. Refer to page 238 to view the table of most commonly used CSS selectors.
  • CSS selectors are case sensitive, so they must match element names and attribute values exactly. Page 292 has more advanced selectors.

How CSS Rules Cascade

Some terms:

  • Last rule: the latter of the multiple identical selectors will take precedence.
  • Specifity: The more specific selector will take precedence over the general ones.
  • !important: You can add !important after any property value to indicate that it should be considered more important than other rules that apply to the same element.

Inheritance

  • Inherited: Specifying font-family for the parent specifies it for the children as well unless you further specify for the child.
  • not Inherited: background color or border properites. Specifying these for the parent will not specify them for the child.
  • NOTE: You can force a lot of properties to inherit values from their parent elements by using inherit for the value of the properites.
