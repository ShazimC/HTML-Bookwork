Chapter 13: Boxes

Box Dimensions

By default a box is sized just big enough to hold its contents. To set your own dimensions you can use the height and width properties.

width: width of box percentages, pixels or ems. Pixels/Percentages are the most popular.
height: height of box in ...
percentages: the size of the box is relative to the size of the browser window or, if the box is encased within another box, it is a percentage of the size of the containing box. This and ems are more useful for designing across devices with different sized screens.

Limiting Width

  • min-width/height: specifies the smallest size a box can be displayed at when the browser window is narrow.
  • max-width/height: indicates the maximum width a box can stretch to when the browser window is wide.
  • NOTE: These are very helpful properties to ensure that the content of pages are legible (especially on smaller screens of handheld devices).

Overflowing Content

  • overflow: tells the browser what to do if the content contained within a box is larger than the box itself. It can have one of two values:
  • hidden: property simply hides any extra content that does not fit in the box.
  • scroll: property adds a scrollbar to the box so that users can scroll to see the missing content.

Border, Margin & Padding

Refer to page 307 for a detailed look.
  • Border: The border seperates the edge of one box from another.
  • Margin: Margins sit outside the edge of the border. You can set the width of a margin to create a gap between the borders of two adjacent boxes.
  • Padding: Padding is the space between the border of a box and any content contained within it. Adding padding can increase the readability of its contents.
  • border-width: is used to control the width of a border. The value of this property can be in pixels or thin, medium, thick. NO PERCENTAGES OR EMS.
    • border-top-width
    • border-right-width
    • border-bottom-width
    • border-left-width
    • border-width: top, right, bottom, left
  • border-style: can control the style of border being used for a box.
    • solid
    • dotted
    • dashed: series of short lines
    • double: two solid lines
    • groove: appears to be carved into the page
    • ridge: appears to stick out from the page
    • inset: appears embedded into the page
    • outset: looks like it is coming out of the screen
    • hidden/none
    • border-top-style, border-left-style..
  • border-color: color of the border
  • border: width, style, and color in one go. ex: border: 3px dotted blue
  • padding: allows you to specify how much space should appear between the content of an element and its border. pixels, percentages, ems.
  • margin: allows you to control the gap between boxes. pixels commonly, percentages, ems. margin: top, right, bottom, left.

Change Inline/Block

display: allows you to turn an inline element into a block-level element or vice versa, and can also be used to hide an element from the page. The values this property can take are:
  • inline: causes a blocklevel element to act like an inline element.
  • block: causes an inline element to act like a blocklevel element
  • inline-block: causes a block-level element to flow like an inline element, while retaining other features of a block-level element.
  • none: this hides an element from the page.

Hiding Boxes

  • visibility: property allows you to hide boxes from users but it leaves a space where the element would have been. This property can take two values:
    • hidden: if the visibility of an element is set to hidden, a blank will appear in its place. If you do not want a blank space to appear, then you should use the display property with a value of none isntead.
    • visible

CSS3: Border Images

  • border-image: applies an image to the border of any box. It takes a background image and slices it into nine pieces. The property requires 3 pieces of information:
    1. The URL of the image.
    2. Where to slice the image.
    3. What to do with the straight edges; the possible values are:
      • stretch: stretches the image
      • repeat: repeats the image
      • round: like repeat but if the tiles do not fit exactly, scales the tile image so they will.

CSS3: Box Shadows

  • box-shadow: allows you to add a drop shadow around a box. It works just like the text-shadow property that you met on page 288. It must use at least the first of these two values as well as a color:
    • Horizontal Offset: Negative values position the shadow to the left of the box.
    • Vertical Offset: Negative values position the shadow to the top of the box.
    • Blur Distance: If omitted, the shadow is a solid line like a border.
    • Spread Of Shadow: If used, a positive value will cause the shadow to expand in all directions, and a negative value will make it contract.
    • Inset (keyword): can be used before these values to create an inner-shadow.
    • box-shadow: horizontal offset, veritcal offset, blur distance, spread of shadow
    • box-shadow: -5px -5px #777777

CSS3: Rounded Corners

  • border-radius: The value indicates the size of the radius in pixels. Can be specified with border-top-right-radius, border-bottom-left-radius etc or border-radius: topleft topright bottomright bottomleft

CSS3: Elliptical Shapes

  • border-radius: To create more complex shapes, you can specify different distances for the horizontal and vertical parts of the rounded corners.
  • border-radius: 80px 50px: will create a radius that is 80px wide and 50px tall in diamter. You can target just one corner with border-top-left-radius..etc
  • Shorthand: first you specify the four horizontal values, then the four vertical values: border-radius: 1em 4em 1em 4em / 2em 1em 2em 1em;
